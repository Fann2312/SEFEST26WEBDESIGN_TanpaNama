# SEFEST26WEBDESIGN_TanpaNama
AION. 
AION adalah website edukatif dan interaktif yang dirancang dan dibuat oleh Raditya Danendra S. & M. Luthfan N., website ini berisi tentang edukasi terhadap lingkungan sekitar manusia, dirancang untuk meningkatkan kesadaran dan pengetahuan masyarakat dengan melalui pendekatan digital yang mudah dipahami atau digunakan, AION menyediakan berbagai fitur keren dan beragam bahkan pengguna dapat menggunakan AION untuk kegiatan sehari-hari!. 
Panduan penggunaan website. 
Pengguna membuka website AION dan menampilkan halaman utama website AION yang berisi pengenalan, lalu ada menu atau menu navigasi yang bertujuan untuk memudahkan pengguna berpindah-pindah halaman website kami atau fitur yang ada di website kami, seperti ada menu edukasi yang bertujuan untuk pengguna membaca materi mengenai lingkungan manusia, lalu ada menu infografis agar pengguna dapat melihat data dengan tampilan visual yang menarik dan bagus agar mudah dipahami.
